# first_2d_mobile_game

Education project from a Udemy course.
This project was initially based on a Udemy course, but it now includes many changes — primarily because the course uses quite old versions of `flame` and `flutter`. In addition, there are numerous improvements on my side, such as a `LinearGradient` health bar, optimized calculation, the use of `mixin`s, and `AssetsGen`.

Some parts of the code are hidden, which may make it a challenge for you. Try to fill in the gaps and run the project after completing all the tasks.
